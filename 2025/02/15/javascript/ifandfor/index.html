IF

삼항 연산자

&&

왼쪽 값이 거짓(false)이면 오른쪽을 실행하지 않습니다.

||

왼쪽 값이 falsy면 오른쪽을 실행합니다.
null, undefined, false, 0, ‘’를 false로 간주합니다.

??

왼쪽 값이 falsy면 오른쪽을 실행합니다.
null, undefined만 false로 간주합니다.

옵셔널 체이닝(?.)

null 또는 undefined이면 오류를 발생시키지 않고 undefined를 반환합니다.

!! 연산자

값을 명확하게 Boolean 타입으로 변환하는 데 사용됩니다.
0, null, undefined, NaN, ‘’을 false로 간주합니다.
