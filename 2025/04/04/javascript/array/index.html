실무에서 자주 쓰는 배열(Array) 문법

  • javascript

검색

indexOf()

조건을 만족하는 요소의 인덱스를 반환합니다.
값이 없으면 -1을 반환합니다.

findIndex()

조건을 만족하는 요소의 첫번째 인덱스를 반환합니다.

find()

조건을 만족하는 첫 번째 요소의 값을 반환합니다.

every()

모든 요소에 대해 조건을 만족하는지 검사합니다.
하나라도 만족하지 못하면 false를 반환합니다.

some()

모든 요소에 대해 조건을 만족하는지 검사합니다.
하나라도 만족하면 true를 반환합니다.

순환

map()

순환 결과를 모아 새로운 배열을 반환합니다.
map은 mapping의 의미를 가지고 있습니다.

filter()

조건을 만족하는 새로운 배열을 만듭니다.

reduce()

순환하여 누적 계산 값을 반환합니다.

변형

slice()

기존 배열을 잘라서 새로운 배열을 만듭니다.
end 인덱스는 포함하지 않습니다.

splice()

배열의 기존 요소를 제거하거나 추가합니다.
기존의 배열을 변경합니다.
(start, deleteCount, items)

join()

배열을 모두 연결해 하나의 문자열로 만듭니다.

push()

배열에 요소를 추가하며 배열의 길이를 반환합니다.

shift()

첫 번째 요소를 제거하고, 그 요소를 반환합니다.

pop()

마지막 요소를 제거하고, 그 요소를 반환합니다.

concat()

배열을 합칩니다.

reverse()

배열을 반전합니다.
