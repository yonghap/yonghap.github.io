실무에서 자주 쓰는 문자/숫자(String/Number) 문법

  • javascript

숫자

toString()

문자를 숫자로 변환합니다.

ceil(), floor(), round()

toFixed()

소수점을 자릅니다.

문자

Number()

문자열을 숫자형으로 바꿔 줍니다.

parseInt()

문자열을 특정 진수의 숫자로 바꿔 줍니다.

+, *

문자열을 숫자로 바꿔 줍니다.

replace()

문자열을 바꿔 줍니다.

replaceAll()

모든 문자열을 바꿔 줍니다.

split()

특정 구분자를 기준으로 문자열을 배열로 만듭니다.

substring()

부분 문자열을 반환합니다.

toLowerCase()

소문자를 반환합니다.

toUpperCase()

대문자를 반환합니다.

tring(), trimEnd(), tringStart()

공백을 제거하고 새로운 문자열을 반환합니다.

padStart()

특정 길이만큼 문자열로 채워줍니다.
