드래그 개념

mousedown

  • 드래그 가능한 요소에 마우스를 눌렀을 때(mousedown) 이벤트 입니다.
  • 드래그 시작한 요소의 복사본 두 개 생성되어, 하나는 원래 위치에 그대로 붙고 다른 하나는 body에 붙습니다.
    (원래 위치에 그대로 붙는 복사본은 옮길 요소의 시각적인 표현 등에 사용됩니다. )

mousemove

  • 마우스를 누르고 움직일 때(mousemove) 이벤트 입니다.
  • 현재 마우스 좌표를 따라 body에 붙은 복사본을 이동 시킵니다.
  • 마우스 좌표가 이동할 박스이거나 이미 드래그된 아이템 위로 옮겨졌을 경우 인디게이터들을 표시해 줍니다.

mouseup

  • 마우스를 떼었을 때(mouseup) 이벤트 입니다.
  • 마우스를 뗀 좌표가 이동할 박스이거나 드래그 된 아이템 위 여부를 체크하여 맞는 위치에 위치시킵니다.
  • 이후 관련 이벤트나 변수 등을 정리합니다.
