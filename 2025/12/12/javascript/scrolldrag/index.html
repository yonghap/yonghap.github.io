ScrollBar(스크롤바) Drag

  • javascript

내부 콘텐츠를 드래그해서 볼 수 있게 해보겠습니다.
traslate로 움직이기도 하지만 scrollLeft, scrollTop을 쓰기도 합니다.

개념

mousedown

  • 드래그 영역을 눌렀을 때 현재 시작 좌표값을 저장합니다.
  • 드래그 상태를 ON 시키고, 시작 아이콘으로 변경합니다.

mousemove

  • 마우스를 누르고 움직일 때(mousemove) 이벤트 입니다.
  • 마우스가 이동한 좌표만큼 스크롤을 이동 시킵니다.

mouseup

  • 마우스를 떼었을 때(mouseup) 이벤트 입니다.
  • 드래그 상태를 OFF 시키고, 종료 아이콘으로 변경 합니다.
