TypeScript : 함수

  • typescript

기본 함수

숫자 매개변수와 숫자 반환값을 가진 함수 입니다.

반환값이 없을 경우 void를 선언 합니다.

종료가 없는 함수의 경우에는 never를 선언 합니다.

함수의 매개변수

선언한 매개변수는 꼭 들어가야 하며, 임의로 추가는 불가능 합니다.
필수가 아닌 매개변수는 ?로 표시 합니다.

매개변수 초기화는 JS 문법과 같습니다.

REST 문법도 활용이 가능합니다.

데이터 호출 결과값으로 객체 배열 데이터도 가능합니다.
모두 같은 기능을 합니다.
